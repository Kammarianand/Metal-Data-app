About This Project

Project Overview

This web application provides real-time gold and silver prices in Indian Rupees (INR). It's designed to give users quick access to current precious metal rates along with additional market data.

Technologies Used

  • HTML5 & CSS3 with Tailwind CSS for styling
  • Vanilla JavaScript for functionality
  • Gold API for real-time data
  • Glassmorphism UI design

API Information

This project utilizes the Gold API (goldapi.io) to fetch real-time metal prices. The API provides:

  • Current prices for gold (XAU) and silver (XAG)
  • Opening prices, daily highs and lows
  • Previous closing prices
  • A troy ounce is a unit of measure used for weighing precious metals, equivalent to 31.1034768 grams.

Features

  • Real-time price updates
  • Detailed market data for both metals
  • glassy user interface
